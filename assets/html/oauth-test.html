📋 Implementation Steps

1

Go to Google Cloud Console

Visit: https://console.cloud.google.com

Create a new project or select existing one

2

Create OAuth 2.0 Credentials

Path: APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client ID

Choose "Web Application" as type

3

Add Authorized Origins & Redirect URIs

JavaScript Origins: http://localhost:8000 http://localhost:3000 http://127.0.0.1:8000 Redirect URIs: http://localhost:8000/assets/html/auth.html http://localhost:8000/callback http://localhost:8000/assets/html/auth.html#google-callback
4

Copy Your Client ID

Format: XXXXX-YYYYY.apps.googleusercontent.com

5

Update auth.js File

File: /assets/js/auth.js

Find line ~5:

const CLIENT_ID = 'YOUR_GOOGLE_CLIENT_ID.apps.googleusercontent.com';

Replace with your Client ID:

const CLIENT_ID = 'your-actual-client-id.apps.googleusercontent.com';
6

Test It Out!

Go to: http://localhost:8000/assets/html/auth.html

Click "Continue with Google" button
